// 02 — ABOUT

I build the backends
you don't notice — and I like it that way.

I spend my days designing event-driven microservices that move money, inventory and orders without anyone noticing. Spring Boot stacks, Kafka topics that don't drop, caches that don't lie, databases that survive 3 AM.

I'm currently a Senior Software Engineer at Sainsbury's working on Argos e-commerce inventory — Kafka Streams, PostgreSQL, DynamoDB, Redis, Caffeine, all stitched together on Kubernetes with GitHub Actions and Terraform.

Before that: Java for Goldman Sachs (via Apexon) on payments rails — RDFI/ODFI, disputes, account flows. Before that: Virgin Money / Clydesdale / Yorkshire Bank via Atos-Syntel, where we hauled legacy banking onto microservices and watched performance jump 80%. Started in Mumbai writing integration tests that nobody else wanted to write.

Outside work: gaming (the kind with steering wheels), tinkering with side projects, and being suspicious of any system labelled "event bus" that's secretly a queue. The drivable car on this page is the gaming and the side-project, combined.
